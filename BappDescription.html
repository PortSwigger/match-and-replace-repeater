The Match & Replace extension enhances the functionality of Burp Suites Repeater tool. Designed for Jython, it adds a specific Match & Replace tab for Repeater requests, permitting pinpoint text replacements within a selected section. Having these targeted alterations remain effective when shifting back to Pretty, Raw, or Hex view increases efficiency and versatility.

Features

  • Integrates a Match & Replace tab within Repeaters Requests.
  • Consists of two fields: a Match field and a Replace field, alongside an Apply Replace button.
  • Enables replacements strictly within the selection — it wont alter anything if no area is selected.
  • If no selection exists initially, it allows users to choose a certain part of the request to modify.

Usage

  1. Select a segment of the request (headers or body) within Repeater.
  2. Open the Match & Replace tab. The request will appear in the native editor.
  3. Insert the text to be searched for in the Match field, and the replacement text in the Replace field.
  4. Click on Apply Replace.
  5. If no selection is available, it will prompt the user to designate a region.
  6. Only the instances within the chosen selection will be replaced.
