Match & Replace (Repeater) adds a dedicated tab to Repeater editors that enables precise text replacements within HTTP requests. This extension supports targeted modifications in specific request regions with regex capabilities and maintains changes across Pretty, Raw, and Hex view modes.

Features

  • Replace text within selected regions, body, headers, URL, or the entire request
  • Regex support with case-insensitive matching options
  • Undo history with up to 10 previous request states
  • Preserves modifications when switching between editor view modes
  • Highlights the modified region after applying replacements

Usage

  1. In Repeater, navigate to the Match&Replace tab
  2. Enter the text to find in the Match field and replacement text in the Replace field
  3. Select a scope from the dropdown: Selection (default), Body, Headers, URL, or Whole request
  4. Optionally enable Regex or Ignore case checkboxes
  5. Click Apply Replace to perform the replacement
  6. Use Undo to revert changes or Clear History to reset the undo stack
